# Kalkulyator OVDP

Калькулятор інвестицій для українських облігацій внутрішньої державної позики (ОВДП), депозитів та інших вкладень.

PWA-додаток, який працює на будь-якому пристрої — Android, iPhone, iPad, десктоп. Встановлюється як нативний додаток, працює офлайн.

## Можливості

**Розрахунки в реальному часі** — результати оновлюються миттєво при введенні даних. Заповніть будь-які відомі поля — решта розрахується автоматично.

**Підтримувані поля:** назва облігації, вартість 1 облігації, кількість, сума вкладення, річна ставка, дати вкладення та отримання, сума отримання, різниця.

**Автоматичні розрахунки:** прибуток, прибуток на 1 облігацію, термін (дні та місяці), дохідність за період, річна дохідність, чистий прибуток.

**Збереження записів** — додавайте кілька облігацій для порівняння. Таблиця з підсумками: загальна сума вкладень, отримань, прибутку та середня річна дохідність. Дані зберігаються між сесіями через IndexedDB.

**Експорт у Excel (.xlsx)** — стилізований файл з темною темою, кольоровим кодуванням (зелений — прибуток, жовтий — дохідність), заголовком та підсумковим рядком.

**Імпорт з Excel** — завантажте раніше збережений файл назад у калькулятор для перегляду та аналізу.

**PWA** — працює офлайн, встановлюється на головний екран пристрою, адаптивний дизайн для мобільних.

## Встановлення

### Веб-версія
Відкрийте в браузері: `https://dzuba1987.github.io/invest-ua/`

### Android
Відкрийте посилання в Chrome → натисніть "Встановити додаток" або меню → "Додати на головний екран".

### iPhone / iPad
Відкрийте в Safari → кнопка "Поділитись" → "На Початковий екран" → "Додати".

### Десктоп (Chrome / Edge)
Відкрийте посилання → натисніть іконку встановлення в адресному рядку.

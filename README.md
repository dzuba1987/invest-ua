# Invest UA

PWA-додаток для управління портфелем вкладень — ОВДП, депозити, облігації та інші фінансові інструменти. Працює офлайн, синхронізується через Google, має Telegram-бот для швидкого доступу.

## Можливості

### Портфель
- Дашборд з вартістю портфеля, денним доходом та прогресом по кожному вкладенню
- Детальна сторінка вкладення з метриками, оподаткуванням та складним відсотком
- Сповіщення про завершення строку вкладень (in-app банери)
- Конвертація вартості портфеля в обрані валюти (НБУ курс)

### Калькулятор
- Розрахунок прибутку, дохідності за період та річної дохідності
- Складний відсоток з графіком порівняння (простий vs складний)
- Ставка реінвестування, індексація (%/рік), горизонт 1-20 років
- Таблиця порівняння по роках
- Історія розрахунків — клік на запис завантажує дані у форму

### Аналітика
- Рейтинг за ефективною річною дохідністю
- Топ-5 комбінацій з оптимальним розподілом бюджету
- Стратегія реінвестування (коротка → довга)

### Валюти
- Офіційний курс НБУ (всі валюти з пошуком)
- Готівковий курс (чорний ринок) з @X_Change_Vinnitsa
- Зміна курсу за день у відсотках
- Вибір 1-2 валют для відображення на дашборді

### Telegram-бот (@my_invest_ua_bot)
| Команда | Опис |
|---|---|
| `/balance` | Баланс портфеля — вкладено, зароблено, дохід/день |
| `/portfolio` | Список усіх вкладень з деталями |
| `/expiring` | Вкладення що завершуються скоро |
| `/rates` | Курси валют НБУ + чорний ринок |
| `/help` | Список команд |

### Інше
- Глобальний пошук по портфелю, історії, валютах
- Хмарна синхронізація через Google (Firebase)
- Excel імпорт/експорт
- PIN-код додатку (SHA-256 хеш)
- XSS санітизація всіх user-даних
- Адмін-панель з Telegram-нотифікаціями при логіні/реєстрації
- Onboarding для нових користувачів

## Встановлення

**Веб:** https://dzuba1987.github.io/invest-ua/

**Android:** Chrome → меню (⋮) → "Встановити додаток"

**iPhone:** Safari → "Поділитися" (□↑) → "На початковий екран"

**Desktop:** Chrome → іконка встановлення в адресному рядку

## Технології

- HTML, CSS, JavaScript (vanilla, без фреймворків)
- Firebase Auth + Firestore (синхронізація)
- Chart.js (графіки)
- xlsx-js-style (Excel)
- Service Worker (офлайн, PWA)

## Бекенд (Laravel)

Окремий мікросервіс [invest-notify](https://github.com/dzuba1987/-invest-notify) для:
- Telegram-бот (webhook, команди)
- Щоденні нотифікації про завершення вкладень
- Проксі курсів валют (чорний ринок)
- Адмін-нотифікації при логіні

## Структура

```
index.html      — основний UI (портфель, калькулятор, аналітика, валюти, профіль)
app.js          — вся бізнес-логіка
styles.css      — темна тема, адаптивний дизайн
i18n.js         — українська + англійська локалізація
config.js       — Firebase та Telegram конфігурація
admin.html      — адмін-панель
help.html       — довідка (v3.0)
sw.js           — service worker
manifest.json   — PWA маніфест
```

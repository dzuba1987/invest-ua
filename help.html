1 Що це за додаток?

Invest UA — PWA-додаток для управління портфелем вкладень: ОВДП, депозити, облігації та інші фінансові інструменти. Працює офлайн, синхронізується через Google.

Портфель
Дашборд з вартістю портфеля, денним доходом та детальною інформацією по кожному вкладенню
Калькулятор
Розрахунок прибутку, дохідності та складного відсотка з графіками
Аналітика
Рейтинг, комбінації та стратегії реінвестування
Валюти
Курси НБУ та готівковий курс (чорний ринок) в реальному часі
Telegram-бот
Баланс, список вкладень, курси валют та сповіщення прямо в Telegram
Хмарна синхронізація
Вхід через Google — дані доступні з будь-якого пристрою

2 Портфель

Головна сторінка додатку. Показує загальну картину ваших вкладень.

Дашборд

Вартість портфеля, зароблено всього, дохід за день, кількість активних вкладень. Нижче — розбивка денного доходу по кожному вкладенню.

Сторінка вкладення

Натисніть на будь-яке вкладення — відкриється детальна сторінка з прогресом, зарахованим доходом, ставкою, оподаткуванням та іншою інформацією.

Сповіщення

Банери на дашборді попереджають коли термін вкладення закінчується. Налаштовується в Профілі (за скільки днів нагадувати).

3 Калькулятор

Розрахунок дохідності окремого вкладення. Заповніть будь-які відомі поля — решта розрахується автоматично.

1
Введіть суму вкладення та річну ставку.
2
Вкажіть дати початку та завершення.
3
Натисніть «Розрахувати» або Enter.
4
«Зберегти запис» — додає до історії розрахунків.

Складний відсоток

Увімкніть чекбокс — з'являться поля:

  • Ставка реінвестування — річна ставка (дефолт 10%)
  • Податок від доходу — % податку на прибуток
  • Індексація — щорічна зміна ставки
  • Горизонт — 1-20 років

Реінвестування відбувається раз на рік. Графік та таблиця порівнюють простий і складний відсоток.

Кредитний калькулятор

Доступний для залогінених. Розраховує ануїтетний платіж: щомісячний платіж, загальну переплату, графік погашення з розбивкою на тіло та відсотки.

Історія розрахунків

Збережені записи відображаються в таблиці. Клікніть на рядок — дані завантажаться у форму для повторного розрахунку.

Поля з блакитним текстом — авторозрахунки. Вручну введене значення стає «фіксованим», решта полів підлаштовуються.

4 Формули розрахунку

Дохідність за період:

Дохідність = (Прибуток / Вкладено) x 100%

Проста річна дохідність:

Річна = Дохідність за період / Днів x 365.25

Складний відсоток (щорічне реінвестування):

Сума = Вклад x (1 + Ставка) ^ Кількість років

Ануїтетний платіж (кредит):

M = P x r x (1+r)^n / ((1+r)^n − 1)
У «Калькуляторі» — проста річна дохідність. У «Аналітиці» — ефективна річна. Складний відсоток враховує щорічне реінвестування прибутку.

5 Аналітика

Доступна після збереження мінімум 2 записів у калькуляторі. Аналізує портфель та пропонує оптимальні варіанти.

$
Бюджет — загальна сума для вкладення.
#
Облігацій у портфелі — скільки різних позицій.
T
Горизонт — фільтр по строку.

Рейтинг — таблиця за ефективною річною дохідністю. Комбінації — топ-5 варіантів розподілу бюджету. Реінвестування — стратегія коротка → довга.

6 Валюти

Курси валют з двох джерел:

1
НБУ (офіційний курс) — Національний банк України. Усі валюти з пошуком.
2
Чорний ринок (@X_Change_Vinnitsa) — купівля/продаж, зміна за день у %, відхилення від НБУ.

Оберіть 1-2 валюти для дашборду (Профіль → Валюти на дашборді). Доступні: USD, EUR, PLN, GBP, CHF, CAD.

7 Сповіщення та Telegram-бот

В Профілі → Сповіщення можна обрати канали:

  • Email — листи на пошту Google акаунту
  • Telegram-сповіщення — нагадування про вкладення
  • Telegram-бот — інтерактивні команди для перегляду портфеля

Команди бота @my_invest_ua_bot

/balance — баланс: вкладено, зароблено, дохід/день

/portfolio — список вкладень

/expiring — що завершується скоро

/rates — курси НБУ + чорний ринок

/help — список команд

Підключення

Профіль → Сповіщення → увімкнути потрібні канали → для Telegram: відкрити бота → вставити Chat ID → зберегти.

8 Імпорт та експорт

Імпорт з Excel: натисніть «Імпорт з Excel» у калькуляторі.

Експорт у Excel: натисніть «Експорт у Excel» під таблицею історії.

9 Глобальний пошук

Поле пошуку зверху сторінки шукає по всьому додатку: вкладення портфеля, історія розрахунків, валюти та навігація по розділах.

10 Встановлення та синхронізація

Хмарна синхронізація: увійдіть через Google — дані доступні з будь-якого пристрою.

Android/Chrome: меню → «Встановити додаток».

iPhone/Safari: «Поділитися» → «На початковий екран».

Desktop: іконка встановлення в адресному рядку Chrome.

Без входу дані зберігаються локально. Google потрібен тільки для синхронізації між пристроями.

11 Безпека

Content Security Policy (CSP) — обмежує джерела скриптів, стилів та з'єднань тільки до довірених доменів.

PIN-код — зберігається як хеш SHA-256, не у відкритому вигляді.

XSS-захист — всі дані користувача екрануються перед вставкою в DOM.

Firebase Security Rules — кожен користувач має доступ тільки до своїх даних.

Telegram Bot Token — зберігається тільки на сервері, не в клієнтському коді.

12 Дані та конфіденційність

Що зберігається: профіль (ім'я, email, телефон), портфель вкладень, історія розрахунків, налаштування.

Де: локально (IndexedDB) та у хмарі (Firebase Firestore) при вході через Google.

Що НЕ зберігається: паролі, платіжні дані, IP-адреси.

Експорт даних: Профіль → Дані та конфіденційність → «Експорт моїх даних» (JSON).

Видалення: Профіль → «Видалити всі дані» — повністю видаляє з пристрою та хмари.

2 Portfolio

Main page of the app. Shows an overview of all your entries.

Dashboard: total value, earned so far, daily income, active entries count. Tap any entry for a detailed view with progress, earnings, and tax info.

Alerts: banners warn when an entry is about to expire. Configure in Profile (days before reminder).

3 Calculator

Calculate returns for a single entry. Fill in any known fields — the rest is auto-calculated.

1
Enter amount and annual rate.
2
Set start and end dates.
3
Click "Calculate" or press Enter.

Compound interest: yearly reinvestment with custom rate, tax, indexation, and 1-20 year horizon. Chart compares simple vs compound growth.

Credit calculator: available for logged-in users. Annuity payments, total overpayment, monthly chart with principal/interest breakdown.

History: saved records in a table. Click a row to reload data into the form.

4 Formulas

Period return:

Return = (Profit / Invested) x 100%

Simple annual return:

Annual = Period return / Days x 365.25

Compound interest (yearly):

Amount = Principal x (1 + Rate) ^ Years

Annuity payment (credit):

M = P x r x (1+r)^n / ((1+r)^n − 1)

5 Analytics

Available after saving at least 2 records. Rankings, top-5 combinations, and reinvestment strategies.

6 Currencies

NBU rates: official rates with search. Black market (@X_Change_Vinnitsa): buy/sell, daily change %, NBU deviation.

Select 1-2 currencies for dashboard: USD, EUR, PLN, GBP, CHF, CAD.

8 Import & Export

Import: "Import from Excel" in Calculator. Export: "Export to Excel" below history. Data export: Profile → "Export my data" (JSON).

9 Global search

Search bar at the top — portfolio, history, currencies, navigation.

10 Installation & sync

Sign in with Google to sync. Install as native app from browser menu. Works offline.

11 Security

CSP headers, PIN stored as SHA-256 hash, XSS sanitization, Firebase Security Rules, bot token server-side only.

Profile → Data & Privacy: see what's stored, export all data (JSON), or delete everything permanently.
